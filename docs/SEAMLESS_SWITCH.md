# 无感换号功能说明

## ✅ 当前方案（合规）：官方登录 + 刷新窗口

点击账号右侧的 **蓝色闪电 ⚡** 后，插件会：

1. 打开 Antigravity 官方登录界面
2. 在登录界面选择账号（插件会提示目标邮箱）
3. 登录完成后自动执行 `Reload Window`，让新账号生效

### 说明

- 账号选择必须由官方登录 UI 完成，插件不会在后台强制切换账号。
- 如果你在登录界面里选错账号，插件会提示“当前登录邮箱 ≠ 目标邮箱”，并让你选择是否仍然刷新窗口。
- 如果取消登录/超时，可能不会自动刷新；可手动执行 `Developer: Reload Window` 后重试。

## 🔧 常见问题

### 1) 点击 ⚡ 没弹出登录

- 先在命令面板确认是否存在 `antigravity.login` 命令。
- 如果没有该命令，可能是 Antigravity 扩展/认证提供方未启用，建议重启 IDE 后再试。

### 2) 切换后仍显示旧账号

- 手动执行一次 `Developer: Reload Window`。
- 再点一次 ⚡，并确认登录界面选择的是目标邮箱。

## 🕘 历史方案（仅供参考）

旧版本曾尝试过内部 API、数据库写入、补丁等方案，易受版本变更影响且风险较高；当前版本不再使用这些方案。
